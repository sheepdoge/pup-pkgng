# pup-pkgng

[sheepdoge](https://github.com/mattjmcnaughton/sheepdoge) pup for managing
dependencies via FreeBSD's `pkg`.

## Variables

`pup-pkgng` expects you to define the following variables:
- `pup_pkgng_packages_to_install`: List of packages to install.
- `pup_pkgng_packages_to_uninstall`: List of packages to uninstall.
